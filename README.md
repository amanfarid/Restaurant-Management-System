# Restaurant Management System

A console-based **Restaurant Management System** developed in **C++**.

---

## Features

- Place orders for **Starters, Main Course, Desserts, and Drinks**.
- **View Cart** and generate final bill with subtotal, tax, service charges, discount, and total amount.
- **Admin panel** for menu management and weekly audit report.
- Generates **Order ID**, date, and time for each transaction.
- Stores session sales for audit purposes.
- Interactive console interface with **colored text** and **sound alerts**.

---

## Technologies Used

- **Language:** C++
- **Platform:** Windows (uses `<windows.h>` for colors and beep)
- **Tools:** Code::Blocks, Visual Studio, or any C++ IDE
